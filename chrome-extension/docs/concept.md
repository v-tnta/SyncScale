# 相対見積もりを用いた、学生向けタスク管理能力認知支援ツール
## 動機
　学生生活において、多くの学生は「〆切に追われる」という心理的負担を抱えている。
これは、単にタスクが多いからではなく、自身の「作業時間の見積もりの甘さ」や「非効率な作業の段取り」が積み重なり、目に見えない「時間負債」が蓄積することも一因である。
　従来のタスク管理ツールでは「〆切の管理」の支援をするが、「見積もりの作業時間と実績の乖離」を可視化する機能は存在しなかった。

## 目的
学生が、時間感覚のズレやタスク規模による先延ばし傾向など「課題に取りかかる際の自身の癖」を客観的に自覚し、メタ認知能力とタスク管理能力を向上させるwebシステムを構築する。

## 手法
PCで扱えるwebアプリ、連携するChrome拡張機能、そしてスマートフォン向けアプリを制作する。
　１．【自動収集】 PCにて、Chrome拡張機能を用いてLMS（manaba等）から課題と〆切データを自動取得し、システムへ登録する。
　2. 【相対見積もり】スマートフォン向けアプリにて、共有された課題に対してラベリングする。(S/M/L)←各レベルが何時間なのかは、ユーザーごとに指定してもらうか、こちらで三段階で指定(すぐ取り掛かれば終わる/1日以内で終わる/数日かかる ( + 何から始めればよいかわかっている / わかっていない))
　3. 【計測と振り返り】 スマートフォン向けアプリで実際に課題に取り組んでいる間、タイマーで実働時間を計測。提出完了時には心身のコンディションを良・中・悪で入力し、振り返る。
　4. 【データの可視化】 収集したログから、ユーザーが定義した「見積もり時間」と「実働時間」の差分（時間負債）、およびタスク規模別の「着手リードタイム（先延ばし度）」をダッシュボード上で視覚化する。

## 結果
時間負債や相対見積もりの誤差をwebアプリ上で一元管理することで、ユーザーが自身の癖や時間の見積もり能力を判断するきっかけとなり、自己管理能力の育成を促す。

## 有効性
　ユーザーの「時間見積もり能力」と「計画立案能力」の向上に直接的に寄与する。
これは学生時代のみならず、社会で必要とされるプロジェクト管理の訓練となる。
漠然とした「焦り」を、分析可能な定量データや図に変換することで、学生の心理的負担を軽減し、データに基づいた自己改善を促す。
また、本研究で得られる「学生の課題管理に最適なUIは何か」という知見は、他の教育支援ツールの開発にも応用可能である。

## 新規性
・タスク管理を通じた自己管理能力の育成支援ツールという位置づけである点
・相対見積もりの誤差を『時間負債』として可視化し、メタ認知を促すアプローチである点
